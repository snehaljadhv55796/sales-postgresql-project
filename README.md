# 🛒 Sales Analytics System (PostgreSQL)

## 📌 Project Overview

This project is a complete Sales Analytics System built using PostgreSQL.
It processes raw CSV data, normalizes it into structured tables, and generates business insights using SQL queries.

---

## 📊 Dataset

The dataset contains sales transaction data including:

* Order ID, Order Date
* Customer Name, Segment
* Product Name, Category
* Sales, Region

---

## ⚙️ Tech Stack

* PostgreSQL
* SQL

---

## 🧱 Project Structure

* `schema.sql` → Table creation and constraints
* `staging.sql` → Raw data import
* `transformation.sql` → Data cleaning & normalization
* `queries.sql` → Business analysis queries
* `views.sql` → Dashboard-ready views

---

## 🚀 How to Run

1. Create database in PostgreSQL
2. Run `schema.sql`
3. Run `staging.sql` to import CSV
4. Run `transformation.sql`
5. Run `queries.sql`
6. Run `views.sql`
